# OOP in Java — Shape Assignment

A simple Java program built for **ITT 06214 – Mobile Application Development**, 
demonstrating the four pillars of Object-Oriented Programming:

- **Abstraction** — `Shape` is an abstract class with an abstract `area()` method.
- **Inheritance** — `Triangle` and `Square` both extend `Shape`.
- **Polymorphism** — Each subclass overrides `area()` with its own implementation.
- **Encapsulation** — `color` is a private field, accessed only through `getColor()` and `setColor()`.

## How to run
Compile and run `Main.java` (it uses `Shape`, `Triangle`, and `Square`).

## Sample output
```
Red shape with area: 12.0
Blue shape with area: 25.0
Square color changed to: Green
```
